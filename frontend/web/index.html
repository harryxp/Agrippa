Help
Agrippa is a keyboard-driven web application that helps the user perform various tasks.

To use it:
  1. Please choose a plugin by typing a keyword, followed by a space.
  2. Plugins give feedback as the user types and are activated when the user presses <Enter>.
  3. Not all plugins need to be activated.
